== Description ==

Bibliar provides a comfortable widget to embed in your website that allow to visits to searching in the Bible. 
Simply add a shortcode to your Wordpress Blog and Bibliar will insert a JavaScript from http://bibliar.com and your users can search in the Bible for words or by versicles.

As a developer you can:
1  Provide a search tool into your website, blog, forums, and so on.
2  Spreading the word of God.

Very simple to install, simple to use. 



== Installation ==

1. Upload `bibliar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode [bibliar] in your page or post

== Frequently Asked Questions ==

= Is Bibliar free of charge ? =

Yes, of course.

= How to use this plugin properly ? =

Intall it. Activate it.
Then use this shortcode in your page or post :

[bibliar]

= Can I show Bibliar in a sidebar widget ? =

Yes, it works in a "text widget" with this same shortcode [bibliar].

= Where is the plugin settings ? =

There is no settings. 

But you can modify the code:

lang="en"  Language in which you prefer in your Widget. (es)  Spanish, (en)  English, (pt)  Portuguese, (it)  Italian

font-size="10"  Size you prefer your Widget.
